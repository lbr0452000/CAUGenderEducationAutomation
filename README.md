# CAUGenderEducationAutomation
중앙대 성평등교육 자동화 프로그램

#### 유의사항
* **본 프로그램은 성평등교육을 편법으로 수강하기 위한 프로그램이 아닌, 수강을 보다 편안히 하기 위한  편의프로그램입니다.** 본 프로그램이 실행되는 동안 교육영상은 순차적으로 정상 재생되며, 프로그램을 중단하면 교육이수 역시 중단됩니다.
* 본 프로그램 사용으로 인하여 발생하는 책임과 불이익은 프로그램 실행자 본인에게 있습니다.
* 사용자 정보는 어디에도 저장되지 않습니다.
